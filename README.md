# Portfolio - B MAA TEJA NAIK

## Overview
This repository contains the source code for a comprehensive professional portfolio and an intelligent educational guidance platform. The project showcases a high-performance web interface integrated with a robust backend architecture, designed to present technical skills, academic achievements, and AI-driven projects effectively.

The platform includes a specialized "One Stop Personalised Career and Education Advisor," an AI-powered solution recognized as a winning entry at the Smart India Hackathon (SIH) 2025.

## Key Features
- **Dynamic Project Portfolio**: Interactive display of full-stack projects with detailed modal views for in-depth exploration of research, technology, and outcomes.
- **Interactive Tech Skills Visualization**: A radial, interactive layout showcasing technical proficiency across various domains including AI, Machine Learning, and Web Development.
- **SIH Winner Showcase**: Dedicated sections for the award-winning Career Advisor project, featuring integrated AI workflows.
- **Responsive Professional UI**: A state-of-the-art design utilizing modern typography, glassmorphism, and optimized CSS for a premium user experience.
- **Backend API Integration**: A FastAPI-powered backend providing structured data for projects and skills.

## Technology Stack

### Frontend
- **Languages**: HTML5, JavaScript (ES6+).
- **Styling**: Vanilla CSS, Tailwind CSS (for utility-first layout optimization).
- **Animations**: GSAP (GreenSock Animation Platform) for smooth transitions and ScrollTrigger integration.
- **Icons**: Lucide Icons for vector-based, crisp iconography.
- **Interactive Components**: Three.js for canvas-based background effects.

### Backend
- **Framework**: FastAPI (Python-based high-performance asynchronous framework).
- **Documentation**: Swagger UI / OpenAPI integration for automated API documentation.
- **Deployment Ready**: Modular structure optimized for containerization or cloud hosting.

## Installation and Local Development

### Prerequisites
- Python 3.8 or higher.
- A modern web browser.

### Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Maateja/portfolio.git
   cd portfolio
   ```

2. Install backend dependencies:
   ```bash
   pip install -r backend/requirements.txt
   ```

3. Run the application:
   ```bash
   python run.py
   ```

The `run.py` script will automatically start both the FastAPI backend server (port 8000) and the frontend HTTP server (port 3000), then open the application in your default web browser.

## Project Structure
- **Root Directory**: Contains the core frontend files (`index.html`, `css/`, `js/`, and `assets/`) for direct deployment compatibility.
- `backend/`: Contains the Python FastAPI application and API endpoint definitions.
- `assets/`: Hosting local media, including project videos, certificates, and professional imagery.
- `run.py`: The unified entry point for local development environments.
